Comment installer la base de données sur un ordinateur en local.

1. Créer une base de données nommée : fotogal
2. Créer un utilisateur nommé : FotoGal_Admin
	2.1 Mettez comme mot de passe : FGEZ_2022
	2.2 Donner lui tout les droits sur la base de données fotogal
3. Utilisé le script sql fourni dans le dossier /CodeSource/DataBase/FotoGal_Dump.sql pour insérer les différentes tables
4. Ouvrez le code source et rendez-vous dans le dossier /CodeSource/models/db_connect.php et vérifiez si les valeurs correspondent bien à celles que vous avez créer précédemment sur dBeaver/phpMyAdmin

	Vous avez installer la base de données, maintenant il vous reste plus qu'a utiliser le site web.
